# Commands for Setting up Repo
## Initialization
```sh
# not needed?
echo ".dot" >> .gitignore

git clone --bare https://github.com/yunglatkes/dotfiles.git $HOME/.dot

# delete conflicting files if present
git --git-dir=$HOME/.dot/ --work-tree=$HOME checkout
```


## Fresh Installation
```
# notice the --bare flag; this initializes the repository as a git bare
git init --bare $HOME/.dot

# add the following to .bashrc
alias dot="/usr/bin/git --git-dir=$HOME/.dot --work-tree=$HOME"

# don't display untracked files in git status
dot config --local status.showUntrackedFiles no
```
